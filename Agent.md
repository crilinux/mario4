# 马里奥风格平台游戏 - 项目规划

## 项目配置
- **素材方案**: 使用颜色块/几何图形代替（后续可替换为真实素材）
- **音效**: 需要添加
- **Boss难度**: 困难（复杂AI，多阶段）

- **项目名称**: Super Platformer Adventure
- **游戏类型**: 2D横版平台跳跃游戏
- **技术栈**: Python + Pygame
- **核心玩法**: 跳跃攻击、躲避陷阱、击败Boss通关

## 2. 游戏设计

### 2.1 玩家角色 (Player)
- **外观**: 卡通风格的管道工角色（类似马里奥）
- **颜色**: 红色帽子、蓝色工作服、棕色靴子
- **尺寸**: 32x48 像素
- **生命值**: 3颗心

### 2.2 操作方式
| 按键 | 功能 |
|------|------|
| ← → | 左右移动 |
| 空格 | 跳跃 |
| F | 攻击（发射火球） |
| ESC | 暂停游戏 |

### 2.3 关卡设计

#### 关卡1: 绿色草地世界
- **主题**: 草地、天空、管道
- **背景**: 蓝天 + 白云 + 绿色山丘
- **地面**: 绿色草地块 + 棕色泥土
- **障碍物**:
  - 问号方块（顶击获得金币）
  - 绿色管道（可进入隐藏区域）
  - 移动平台
- **敌人**: 蘑菇怪、乌龟
- **Boss**: 板栗仔（大型版本）- 困难模式
  - **血量**: 8
  - **技能**:
    - 冲撞攻击（快速向玩家冲刺）
    - 跳跃砸击（跳起后下落砸向玩家）
    - 召唤小怪（每隔一段时间召唤2个小蘑菇）
    - 护盾阶段（血量低于50%时，进入3秒无敌）
    - **AI行为**: 根据玩家位置动态选择攻击方式，有攻击冷却

#### 关卡2: 地下洞穴/岩浆世界
- **主题**: 黑暗洞穴、岩浆、砖块
- **背景**: 深蓝色天空 + 星星 + 暗色洞穴
- **地面**: 棕色砖块 + 金属平台
- **障碍物**:
  - 暗道门
  - 上升/下降平台
  - 岩浆池（碰到扣血）
- **敌人**: 蝙蝠、岩浆怪
- **Boss**: 酷霸王 - 困难模式
  - **血量**: 12
  - **技能**:
    - 远程火球（发射追踪火球）
    - 瞬移（随机传送到场地随机位置）
    - 冲刺攻击（蓄力后快速多段冲刺）
    - 召唤岩浆（场地随机位置喷发岩浆）
    - **阶段变化**:
      - 阶段1 (100%-66%): 基础攻击
      - 阶段2 (66%-33%): 加速攻击 + 召唤岩浆
      - 阶段3 (33%-0%): 全技能联动 + 增加移动速度

## 3. 技术架构

```
mario_game/
├── main.py              # 游戏主入口
├── constants.py         # 常量定义
├── player.py            # 玩家类
├── enemy.py             # 敌人基类 + 小怪
├── boss.py              # Boss类
├── bullet.py            # 子弹/火球类
├── level.py             # 关卡管理
├── platform.py          # 平台/地面类
├── item.py              # 道具（金币、心）
├── assets/             # 资源文件夹
│   ├── sprites/         # 精灵图片
│   └── sounds/          # 音效
└── levels/              # 关卡数据
    ├── level1.tmx      # 关卡1数据
    └── level2.tmx      # 关卡2数据
```

### 核心类设计

#### Player类
```python
class Player(pygame.sprite.Sprite):
    - move(dx): 水平移动
    - jump(): 跳跃
    - attack(): 发射火球
    - update(): 更新状态
    - draw(): 绘制
```

#### Boss类
```python
class Boss(pygame.sprite.Sprite):
    - take_damage(amount): 受伤
    - attack(): 执行攻击
    - ai_update(): AI行为
    - move(): 移动模式
```

## 4. 游戏流程

```
开始界面
    ↓
关卡1选择 → 游玩关卡1 → Boss战(板栗仔) → 胜利 → 关卡2
    ↓
关卡2 → Boss战(酷霸王) → 胜利 → 游戏通关 → 结束界面
```
